Endringer

Ikke skalering av bildet av hund på hjemmeside, men skaleringen gjøres på hundebildene under our_dogs.html. Kun validering av telefonnummer ved ordreskjema og ikke for alle inputs.

Løsning av minimum krav

Sander implementerte sitt java krav med click-on funksjon for hundebildene på our_dogs.html. Marcus og Noah implementerte funksjonaliteten om filter for visning av hunder basert på brukerdefinerte egenskaper. Ove og Morten fullførte sitt minstekrav tilknyttet validering av ordreskjema i dokumentet eksjava.js Sondre fullførte oppgaven med pop-up vindu som takker for bestillingen etter innsendelse av gyldig ordreskjema.

Arbeidsfordeling

Følgende arbeidsfordeling er gjort for html dokumentene: Marcus skrev dokumentet for hjemmesiden. Noah gjorde siden som viser frem hundene. Morten fullførte siden for ordreskjema. Sander fullførte siden for informasjon om virksomheten. Ove implementerte dokumentet for tilbakemeldingsskjema. Sondre tok seg av CSS dokumentet, mens alle samarbeidet om javascript funksjonene.

Modularitet

Fyll inn mere når ferdig ...


Vi har løst modularitetskravet med å sette en felles javascript fil som tar inn banner, navigation-baren og footeren for alle de ulike sidene av nettsiden. Dette ble gjort av Morten og Ove.
